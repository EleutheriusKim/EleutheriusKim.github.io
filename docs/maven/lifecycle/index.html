LifeCycle

  1. pre-clean
  2. clean
  3. post-clean

default LifeCycle#

  1. validate : 프로젝트가 빌드를 진행 할 수 있는 정보에 대한 유효성 검사
  2. initialize : 빌드 상태를 초기화. (properties 설정 및 directories 생성)
  3. generate-sources : 컴파일에 포함되는 모든 소스코드를 생성
  4. process-sources
  5. generate-resources : 패키지에 포함할 리소스를 생성
  6. process-resources : 리소스를 목적지 디렉토리에 복사 및 처리하여 패키징 할 준비
  7. compile : 프로젝트의 소스코드를 컴파일
  8. process-classes : 컴파일되어 생성된 파일들의 후처리. (ex. Java 클래스에서 Bytecode Enhancement)
  9. generate-test-sources
  10. process-test-sources
  11. test-compile
  12. process-test-classes
  13. test
  14. prepare-package
  15. package
  16. pre-integration-test
  17. integration-test
  18. post-integration-test
  19. verify
  20. install
  21. deploy

  1. pre-site
  2. site
  3. post-site
  4. site-deploy
