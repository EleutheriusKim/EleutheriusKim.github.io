유용한 HTML 태그

September 5, 2021
hwayoung kim
소주같은 개발자

<details/> + <summary/>#

  • 클릭 시, 내용을 보여주는 toggle을 지원
  • 필자는 <details/>를 사용하여 참조 URL을 작성
<details>    <summary>Click Me!!!</summary>    <div>        <ul>            <li>목록1</li>            <li>목록2</li>            <li>목록3</li>        </ul>    </div></details>
Click Me!!!
  • 목록1
  • 목록2
  • 목록3

<kbd/>#

  • keyboard 입력을 표현할 때 사용됩니다
<p>Press <kbd>Ctrl</kbd> + <kbd>C</kbd> to copy text (Windows).</p><p>Press <kbd>Cmd</kbd> + <kbd>C</kbd> to copy text (Mac OS).</p>

Press Ctrl + C to copy text (Windows).

Press Cmd + C to copy text (Mac OS).
