package.json 업데이트 확인

August 27, 2021 · One min read
hwayoung kim
소주같은 개발자

소개#

npm-check-updates라는 모듈이 있다.

해당 모듈을 사용하면 package.json 내에 등록된 모듈들을 의존성에 맞게 최신버전으로 업데이트가 가능하다. npm -g를 사용하여 global로 설치하여 사용하면 편하게 사용할 수 있다.

  • https://www.npmjs.com/package/npm-check-updates

설치방법#

npm i -g npm-check-updates

사용법#

## version Update Checkcd project-root-pathncu

## version Updatecd project-root-pathncu -unpm install
